ZIP Structure

A playlist is imported as a ZIP file containing a CSV file and media files. The ZIP must include:

  • playlist.csv: Defines the playlist name and song details.
  • Audio Files: One audio file per song (e.g., song.mp3).
  • Art Files: One image file per song for album art (e.g., cover.jpg).

All files must be at the root of the ZIP (no subfolders). Example:

my_playlist.zip

CSV Format

The playlist.csv file specifies the playlist and its songs. It must have:

  • First Row: Playlist Name, (e.g., Playlist Name,My Favorites).
  • Second Row: Title,Artist,AudioFile,ArtFile (exact header).
  • Song Rows: One row per song with four columns, all in double quotes (") and separated by commas.
Column Description Example
Title Song title. "Song, One"
Artist Artist name (can be empty). "Artist A" or ""
AudioFile Audio filename in the ZIP. "song.mp3"
ArtFile Art filename in the ZIP. "cover.jpg"

Songs appear in the playlist in CSV order. Example:

Playlist Name,My Favorites
Title,Artist,AudioFile,ArtFile
"Song, One","Artist A","song.mp3","cover.jpg"
"Track Two","","track.mp3","artwork.png"

Requirements:

  • All song row fields must be in double quotes.
  • Fields must be separated by commas.
  • Title, AudioFile, and ArtFile must have values; Artist can be empty ("").
  • The header must be Title,Artist,AudioFile,ArtFile.

File Naming

Audio and art files can have any valid filename, as long as they match the AudioFile and ArtFile columns in the CSV exactly.

Type Formats Notes
Audio MP3, WAV, OGG, AAC MP3 is recommended.
Art JPG, PNG, GIF JPG or PNG is recommended.

Requirements:

  • Filenames in the CSV must match the ZIP files, including extensions.
  • Each audio and art file must have a unique name in the ZIP.
  • Avoid special characters; spaces are okay but not recommended.

Creating a Playlist

Follow these steps to create a playlist ZIP for Dreamify:

  1. Make the CSV:
    • Use a text editor or spreadsheet software.
    • Follow the CSV Format requirements.
    • Save as playlist.csv.
  2. Get Media Files:
    • Obtain audio and art files from legal sources.
  3. Name Files:
    • Ensure filenames match the CSV‚Äôs AudioFile and ArtFile columns.
  4. Create the ZIP:
    • Add playlist.csv and all audio/art files to a ZIP at the root.
    • Name the ZIP (e.g., my_playlist.zip).
  5. Import:
    • In Dreamify, click Import Playlist, select the ZIP, and wait for the playlist to appear.

Extra Information

Testing Playlists

  • Export a playlist from Dreamify to see the correct ZIP and CSV format.
  • Test with a small ZIP (1‚Äì2 songs) before creating large playlists.

File Sizes

  • Use compressed audio (e.g., MP3) and smaller images (e.g., JPG/PNG) to speed up imports.
  • Large files may slow imports or hit browser storage limits.

Common Import Errors

  • Missing playlist.csv: Ensure it‚Äôs at the ZIP‚Äôs root.
  • Incorrect CSV format: Check the first row is Playlist Name,, the header is Title,Artist,AudioFile,ArtFile, and all song fields are quoted.
  • Missing files: Verify all audio/art files in the CSV are in the ZIP with exact names.
  • Subfolders: Remove any subfolders; all files must be at the root.
